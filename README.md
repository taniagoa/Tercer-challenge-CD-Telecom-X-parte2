# Tercer-challenge-CD-Telecom-X-parte2
Tercer challenge de ONE G9 Analisis de evasion de clientes

## El objetivo es desarrollar modelos predictivos capaces de prever qué clientes tienen mayor probabilidad de cancelar sus servicios.
Usaremos los datos que procesamos en el desafio anterior, dividiremos el trabajo en objetivos mas pequeños que son:<br>
-Realizar análisis de correlación y selección de variables.<br>
-Entrenar dos o más modelos de clasificación.<br>
-Evaluar el rendimiento de los modelos con métricas.<br>
-Interpretar los resultados, incluyendo la importancia de las variables.<br>
-Crear una conclusión estratégica señalando los principales factores que influyen en la cancelación.
